abstract: We propose a cascaded learning framework for optical power spectrum prediction across multi-span fibre links that propagates prediction uncertainty end-to-end using Noisy Input Gaussian Processes (NIGPs). By treating the output of each span model as a noisy input to the next, the approach accounts for uncertainty accumulation along the link. Applied to a 5-span experimental link with 6 EDFAs, the NIGP cascaded model achieves a mean absolute error of 0.12 dB while requiring substantially less training data than standard deep neural network approaches.
